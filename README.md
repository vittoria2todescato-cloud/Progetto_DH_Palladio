# Progetto_DH_Palladio
Il mio progetto consiste nella raccolta catalogata di 10 ville palladiane presenti nella provincia di Vicenza. In particolare ho sviluppato la pagine di Villa Almerico Capra (detta la Rotonda) descrivendonone i metadati e presentando un estratto del II libro del tratto "I quattro libri dell'architettura" di Andrea Palladio. 
Il sito è navigabile attraverso un catalogo (lista), una mappa dei luoghi, una linea del tempo e uno schema sugli usi attuali delle ville.
